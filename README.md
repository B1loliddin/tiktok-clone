## Features

- Sign in/Sign up
- Uploading Videos with Caption
- Compressing Videos
- Generating Thumbnails Out of Video
- Displaying Videos with Caption
- Liking on Posts
- Commenting on Posts
- Liking the Comments
- Searching Users
- Following Users
- Displaying Followers, Following, Likes
- TikTok Like UI

## Installation

After cloning this repository go to `tiktok-clone` folder. Then, follow the following steps:

- Create project in [firebase](https://firebase.google.com)
- Connect your firebase to the project

## Packages used

- get
- firebase_core
- firebase_auth
- firebase_storage:
- cloud_firestore
- image_picker
- video_player
- video_compress
